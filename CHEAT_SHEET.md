# Cheat Sheet

| Command       | Description   |
| ------------- |:-------------:|
| ..            | Go up one directory |
| ...           | Go up two directory |
| ....          | Go up three directory |
| -- -          | Go back |
| l             | List files |
| ll            | Pretty list files |
| fn            | Find files |
| hi            | History |
| d             | To dropbox |
| reload        | Reload .bashrd |
| editgit       | Edit .gitconfig |
| editbash      | Edit .bashrc |
| h             | Go tom ~/ 'home directory' |
| s[.]          | Open in Sublime Text |
| a[.]          | Open in Atom.io |
| g             | git |
| ga            | git add |
| gau           | git add -u |
| gl            | git log |
| gs            | git status |
| gss           | git status --short |
| gd            | git diff |
| gds           | git diff --staged |
| gdt           | git difftool |
| gdts          | git difftool --staged |
| gm            | git commit -m |
| gma           | git commit -am |
| gb            | git branch |
| gc            | git checkout |
| gra           | git remote add |
| grr           | git remote rm |
| gpu           | git pull |
| grh           | git reset HEAD |
| undo          | git reset --soft HEAD^ |
| gup           | git up //Fetches all branches |
| gcommit       | Commit and push to git |
| gpull         | Pull all branches and update submodules |
| gmerge        | Merge a to b and push |
| chrome        | Open in url in Chrome |
